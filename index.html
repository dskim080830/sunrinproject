'경로 계산' 버튼을 누르면 자동으로 서울 열린데이터광장 API를 먼저 불러온 뒤 경로를 계산합니다. 이 버튼은 데이터만 수동으로 다시 불러오고 싶을 때 눌러주세요.

엘리베이터 고장 시뮬레이션

역을 클릭하면 '엘리베이터 고장' 상태로 표시됩니다. 실제 서비스에서는 공공데이터포털의 역사 편의시설 실시간 API가 이 목록을 자동으로 갱신합니다 — 지금은 데모를 위해 수동으로 토글합니다.

알고리즘 동작 방식 — 지하철역 전체를 그래프의 정점(node)으로, 인접한 두 역 사이의 선로를 간선(edge)으로 표현합니다. 출발역에서 BFS를 수행하며 elevatorOk = false인 역은 탐색 대기열(queue)에 아예 진입시키지 않아, 최단 정차역 수를 보장하면서도 엘리베이터가 없는 역을 원천적으로 우회합니다. 환승은 같은 역에 여러 호선이 겹치는 지점으로 자동 판정됩니다.
